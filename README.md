## LAPORAN PRAKTIKUM 7

## Tujuan Praktikum
Tujuan dari praktikum ini adalah untuk menerapkan konsep dasar HTML, CSS, dan JavaScript dalam membangun sebuah website interaktif,
dengan studi kasus pembuatan sistem Toko Buku Online yang memiliki beberapa fitur seperti login, katalog buku, pemesanan, 
dan pelacakan pengiriman.

## Dasar Teori
Dalam pembuatan website ini digunakan tiga komponen utama pemrograman web, yaitu:
### 1) HTML (HyperText Markup Language)
Digunakan untuk membuat struktur dasar halaman web seperti form, tabel, dan navigasi.

### 2) CSS (Cascading Style Sheets)
Digunakan untuk mengatur tampilan, warna, font, dan layout agar website lebih menarik serta responsif.

### 3)JavaScript
Digunakan untuk menambahkan interaktivitas dan logika, seperti validasi login, penambahan data dinamis, dan simulasi pemesanan.

### 4)Konsep Data Client-Side
Semua data pengguna, katalog buku, dan tracking disimpan sementara pada file JavaScript (data.js) dan ditampilkan secara dinamis ke halaman web.

## KESIMPULAN
Proyek website Toko Buku Online ini berhasil dibuat sebagai simulasi sistem penjualan dan pemesanan buku secara online menggunakan kombinasi HTML, CSS, dan JavaScript. Seluruh fitur utama telah terintegrasi dengan baik, mulai dari halaman Login untuk autentikasi pengguna, Dashboard sebagai pusat navigasi, Stok / Katalog Buku untuk menampilkan dan menambah data buku, Checkout untuk melakukan pemesanan dengan perhitungan total harga otomatis, hingga Tracking Pengiriman yang menampilkan status perjalanan pesanan. Dari sisi tampilan, website menggunakan tema hijau yang konsisten untuk menciptakan kesan bersih dan profesional, dengan desain yang responsif dan mudah digunakan. Keseluruhan proyek ini menggambarkan penerapan dasar pemrograman web secara utuh, meliputi pengelolaan data, interaksi pengguna, serta penerapan estetika visual melalui CSS, sehingga dapat dijadikan dasar pengembangan menuju sistem toko buku online yang lebih lengkap di masa mendatang.
